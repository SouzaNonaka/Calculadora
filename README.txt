Calculadora utilizando React com TypeScript, utilizando CSS puro para estilizar. Também foi usado funções nativas do js.
o processo foi focado em ui/ux, lógica e robustez e por fim refino na interatividade, dando suporte total ao teclado.

Como rodar? Clone o repositório, instale as dependências (npm install) e inicie o servidor (npm run dev).
